The game consists of a simulation where the player takes the place of a farmer who must try to repair his land due to the contamination caused by industrial waste. The simulation runs through a field that is divided into quadrants that the player must repair through 3 games that allow him to recreate each of the treatments for the symptoms mentioned above.

The player has nearby elements with which they must interact in a specific order to successfully overcome the simulation. The game map has a field that is divided into 15 quadrants that the player must partially repair in each of the 3 games so that in the end, it is completely healthy.





First of all, for the first game each quadrant is shown with a sign indicating their current PH level, which is in a range between 0 and 6, the player must take the transparent bottle of the month and go to each of the quadrants where you must pour the liquid from the bottle over each one, when doing so the PH number will begin to increase with a random speed. The objective is to bring the number to a value that is between 7 and 8, in case of exceeding the range, the ground will be completely damaged and it will not be possible to continue repairing it in the other stages.

When advancing to the second game, on the quadrants that have been successfully saved in the first game a message will appear with the word "Nurture!", this message will appear in a random order on the quadrants and the player must take the bucket and deposit the content in each of the quadrants as the message appears. In the case of the second game mode (explained later in this section), the "Nurture" message will appear for a few seconds and will disappear, in case the player fails to reach the quadrant before the message disappears, the quadrant in question will be irreparable.
